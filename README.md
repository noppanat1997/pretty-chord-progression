# pretty-chord-progression

## All Commands

### Default target

```
make all
```

### Build target

```
make build
```

### Run target

```
make run
```

### Clean target

```
make clean
```

### Help target

```
make help
```

## Example Input File

```
YOASOBI - Gunjo [Chord Progression]

(1)
Ebmaj7 Ebmaj7 D7,D7/F# Gm
Ebmaj7 D7,D7/F# Gm Bb/Ab,Bb
Ebmaj7 F7,F7/Eb D7,D7/F# Gm,Bb
Ebmaj7 F

(A)
Ebmaj7 Ebmaj7 Bb B
Ebmaj7 F Bb D7/F#,D7
```
